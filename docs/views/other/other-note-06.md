---
title: win终端添加gitBash
---

1. 打开windows terminal;

2. 点击设置;

3. 打开JSON文件，修改配置；

   - 找到 `profiles–>list`

   ```json
   {
   	"commandline" : "C:\\Program Files\\Git\\bin\\bash.exe"                ,
   	"guid" : "{f263b0f3-4e4d-d630-298c-605384d8fa48}",
   	"icon" : "C:\\Program Files\\Git\\mingw64\\share\\git\\git-for-windows.ico",
   	"name" : "Git Bash"
   }
   ```
